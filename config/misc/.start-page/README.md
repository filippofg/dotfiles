## Shortcuts

Default provider: DuckDuckGo

- `:g` → search on Google
- `:w` or `:we` → search on Wikipedia (en)
- `:wi` → search on Wikipedia (it)
- `:aw` → search on ArchWiki
- `:dw` → search on Debian Wiki
- `:so` → search on StackOverflow
